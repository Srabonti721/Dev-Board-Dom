Question-3: What is event delegation in the DOM?

Event delegation is a JavaScript design pattern that uses a single event listener on a parent element to manage events for all its child elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is a mechanism in the DOM where an event triggered on a child element propagates upward, firing the same event on its parent and ancestor elements, all the way to the root.
